    Application for Income Recording

    Features

    • Application allows creating, editing and removing Employees and Work Types
    • Users can produce and store Working sheets using interactive forms
    • Tool for automatic producing invoices is provided
    • The working history is visible through integrated browser which is enhanced by set of filters

    Technologies

    • We use Apache Maven Server
    • Database layer is from noSQL family (XML)
    • Css stylesheets are produced by LESS preprocessor
    • Front-end processes are mostly handled by jQuery library

    Here is downloadable PowerPoint presentation Wireframe (.pptx). We used integrated Storyboarding functionality to design set of user interface pages.
